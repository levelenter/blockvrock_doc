#  機能説明
3D空間を表すブロックです。プロジェクトに1つ必要で、複数は設定できません。
"シーンスタート"との違いはライブラリや<a-scene>に付属されている属性などが設定されておらず、単純な構成となっています。

"画像追跡ARシーン"や"顔認識AR"は通常の"シーンスタート"では動かないので"ARシーンスタート"を使ってください。
